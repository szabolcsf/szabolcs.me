# Year of the Linux Desktop : we are there

Memes aside, I think this is it: 2025, we are finally there. Or very close. [Debian Trixie](https://www.debian.org/News/2025/20250809) is working out-of-the-box on a [new laptop](https://www.hp.com/us-en/workstations/zbook-ultra.html) with Ryzen AI 395+ Pro / Radeon 8060s. Default kernel. Default mesa. Nothing compiled by hand or installed from 3rd party repos. It can run [Ollama](https://ollama.com/) and a bunch of different models locally at really good speeds, although for a recent RoCM, you'll need a Docker or Podman container with Ubuntu.

Okay, okay, I admit, too good to be true: the webcam is not working, support is coming(?) to mainline Linux with [6.18](https://www.phoronix.com/news/AMD-ISP4-Driver-Pending-Review). Or one could go the Ubuntu + OEM kernel route. But for me, who frantically looks for the hardware kill-switch to turn off the camera, it's a non-issue.

In other, related news: I think [Hyprland](https://hypr.land/) is really cool these days, especially if you pair it with Quickshell. There are a bunch of great dotfiles repos if you don't wanna start from scratch. I like [JaKoolit's repo](https://github.com/JaKooLit/Debian-Hyprland) these days, because it does the setup for you, and it supports Debian (as well as [Ubuntu](https://github.com/JaKooLit/Ubuntu-Hyprland) and [Arch](https://github.com/JaKooLit/Arch-Hyprland)). 

Other notable mentions in this area:
 - [DankMaterialShell](https://github.com/AvengeMedia/DankMaterialShell) : dots
 - [end-4](https://github.com/end-4/dots-hyprland) : dots
 - [Omarchy](https://omarchy.org/) : Arch + dots

Now imagine this: running your LLM locally, you can do [ricing](https://jie-fang.github.io/blog/basics-of-ricing) with low effort. Just think about the possibilities!
